Har fixat en on click funktion i koden, vet inte exakt hur jag kommer att använda den men just så försvinner allt då man klickar på något. vill göra så att man går vidare till olika ställen beroende på vad man klickar på. Det är något för nästa gång.

{#if start_screen}
	<div class="news-container">
		{#each articles as article, i}
		<div
			on:click={() => {
			start_screen = false;
			}}
			on:keypress={() => {
			start_screen = false;
			}}
			
			class ="news"
		>
			<img src={articles[i].picture} alt="" />
			<p>{articles[i].titel}</p>
		</div>
		{/each}
	</div>
{/if}
